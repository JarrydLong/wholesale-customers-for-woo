Allow wholesale customers for your WooCommerce store. Currently supports global discount.

== Description ==
Customers can buy for wholesale pricing in your WooCommerce store. Offer certain users global discounts for wholesale price (more features to be added frequently).

== Installation ==
1. Upload the plugin files to the \'/wp-content/plugins\' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress.
3. Navigate to \'WooCommerce\' and data will be showing under \'General Settings Tab\' > \'Pricing Options\'.
4. To enable users to become wholesalers, simply enable the \'Wholesale Customer\' checkbox in the \'Edit User Profile\'.
